# MIS Galahad 10 PE
本PE基于Windows 10 20H2 64位系统镜像制作。


特点 Features：
1. 内置网卡驱动，可联网；
2. 浏览器：Microsoft Edge Chromium；
3. 支持HTTP、 HTTPS、 FTP 和 BT的下载器：Free Download Manager；
4. WinNTSetup；
5. 分区工具：傲梅分区助手；
6. 备份工具：傲梅备份软件；
7. 压缩文件查看器：7-zip；
8. 系统引导工具：Bootice；
9. 硬件识别：CPU-Z；
10. 测速工具：Speccy；
11. 文件搜索：Everything；
12. 恶意软件删除工具：McAfee Stinger，Avira PC Cleaner；
13. 图片查看器：Irfanview；
14. 音视频播放器：VLC media player
