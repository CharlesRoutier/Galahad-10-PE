1. 内置网卡驱动，可联网；
2. Microsoft Edge Chromium：新版Microsoft Edge浏览器，基于Chromium内核。更新到最新版（内部版本号89.0.767.0）；
3. Free Download Manager：一款开源且功能强大的免费下载器，支持用户通过 HTTP、 HTTPS、 FTP 和 BitTorrent从远程服务器下载文件。
